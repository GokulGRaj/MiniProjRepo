# MiniProjRepo
Details about a dashboard created using Power BI

Disney Plus Shows Analysis
1. Project Overview
This project analyses Disney Plus shows and movies data using Excel and Power BI to uncover trends in content distribution, genres, release patterns, and ratings. The goal is to provide insights into the platform’s content strategy and help understand audience preferences.
2. Tools Used
•	Microsoft Excel
•	Power BI
3. Dataset
•	Source: Sample dataset (Disney Plus dataset from public domain / Kaggle)
•	Data Contains:
o	Imdb ID
o	Title
o	Plot 
o	Type (Movie/Series)
o	Rated
o	Year
o	Released At
o	Added At
o	Run time
o	Genre
o	Director
o	Writer
o	Actors
o	Language
o	Country
o	Award
o	Meta Score
o	Imdb rating
o	Imdb votes
4. Steps Followed
•	Data Cleaning in Excel:
o	Removed duplicate records
o	Handled missing/null values
o	Standardized column formats and datatypes (date, text)
o	Split and formatted columns (e.g., genres, actors, language,writer)
•	Data Analysis in Excel:
o	Used basic formulas and lookup functions for validation
o	Created Pivot Tables for initial insights
•	Power BI Implementation:
o	Imported cleaned dataset into Power BI
o	Created relationships (if multiple tables used)
o	Built calculated columns and measures (DAX)
•	Dashboard Creation:
o	Created visuals like bar charts, pie charts, and line charts
o	Added slicers for filtering (Year, Genre, Country, Type)
o	Designed KPI cards for key metrics
5. Key Insights
•	The majority of content on Disney Plus is Movies compared to Series
•	Most content was added to the platform after 2015, showing recent expansion
•	Family, Animation, and Adventure are the most dominant genres
•	The United States contributes the highest number of shows
•	Certain ratings like G and PG dominate, indicating family-friendly content
•	Content addition peaked in specific years, indicating platform growth phases

6. Conclusion
The analysis highlights Disney Plus as a family-focused streaming platform with a strong emphasis on movies and recent content additions. The insights can help in understanding content trends and guiding future content strategies.
